Custom Gutenberg blocks and an events system for WordPress.

== Description ==

Buttercup provides custom Gutenberg blocks and a full-featured events system for WordPress sites.

**Blocks included:**

* **Row Layout / Row Column** -- Flexible row-based layouts for organizing content.
* **Homepage Feed** -- A dynamic homepage feed with per-item rendering and auto-expand.
* **Tag Showcase** -- Display posts by tag in a visual showcase.
* **Team / Team Member** -- Showcase team members with dedicated container and member blocks.
* **Events Archive** -- Display upcoming and past events in an archive view.

**Events system:**

* Custom post type for events with start date, end date, and location fields.
* Facebook event sync via the Graph API on a WP-Cron schedule.
* iCal feed import support.
* Dedicated single and archive templates.
* Settings page for managing event defaults and display options.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/buttercup` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the new blocks in the block editor under the Buttercup category.
